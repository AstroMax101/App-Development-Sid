Desktop Apps:
  • Desktop apps are developed to run on desktop, computer and laptop.
  • They are developed for a particular operating system.
  • Example : Word processing, Spreadsheet and Calculator.


Web Apps:
  • Web apps are the apps which run on web browser.
  • Internet connection is mandatory for the web page to run.
  • Example : Google Maps, Facebook and Twitter.


Mobile Apps:
  • Developed for mobile devices and are designed to run on mobile devices like Smartphone and tablet.
  • Mobile devices have operating system like android, iOS, Blackberry OS, Symbian and Windows Phone OS.
